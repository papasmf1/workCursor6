## 🚀 시작하기

이 저장소의 자료들을 통해 바이브 프로그래밍의 다양한 도구와 기법을 학습할 수 있습니다.

### 주요 학습 내용

1. **코딩 어시스턴트 활용** - AI 기반 코딩 도구 사용법
2. **Cursor IDE** - 현대적인 코드 에디터 활용
3. **VS Code + 코파일럿** - Microsoft의 AI 코딩 어시스턴트
4. **클라우드 코드** - 클라우드 기반 개발 환경
